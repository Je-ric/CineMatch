# üé¨ CineMatch (Movie Recommendation System)

CineMatch is a movie discovery platform built with **Laravel**, **Livewire**, and **Tailwind CSS**.  
It allows users to explore movies, manage favorites, leave reviews, and receive personalized recommendations based on their activity (favorites and rated & reviewed movies).

---

## üöÄ Features

### üîê User Authentication
- Email/username login & registration  
- Social login with **Google** and **Facebook** (Laravel Socialite)

### Movie Database
- Browse movies with full details  
- Genres, cast, directors, ratings  
- Dynamic movie pages powered by Livewire components

### Favorites System
- Add or remove movies from favorites  
- Tracks genre preferences automatically

### Ratings & Reviews
- Leave star ratings and text reviews  
- Real-time updates using Livewire  
- Displays average rating and review list

### Personalized Profile
- Favorites tab  
- Rated movies tab  
- Smart recommendations based on user activity

### Movie People Management
- Add/remove directors and cast  
- Livewire-powered dynamic interface
